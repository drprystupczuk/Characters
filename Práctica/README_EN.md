# Academic Project - Characters Viewer

This is a small Windows Forms academic project created for learning purposes.

Description:
- A main window with buttons to open the Characters viewer and a Credits window.
- The Characters form shows an image for a selected item from a drop-down.
- The Credits form shows author information.

All UI elements and code identifiers have been updated to English for clarity.
