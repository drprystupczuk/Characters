# Proyecto Académico - Visor de Personajes

Este es un pequeño proyecto de Windows Forms realizado con fines académicos.

Descripción:
- Ventana principal con botones para abrir el visor de Personajes y la ventana de Créditos.
- El formulario de Personajes muestra una imagen según la selección en un desplegable.
- El formulario de Créditos muestra la información del autor.

Todos los elementos de la interfaz y los identificadores de código se han actualizado al inglés para mayor claridad.
